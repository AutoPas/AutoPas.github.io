Detailed Description

template<class Particle, class NeighborList>
class autopas::VarVerletLists< Particle, NeighborList >

Variable Verlet Lists container with different neighbor lists.

Template Parameters
ParticleThe particle type this container contains.
NeighborListThe Neighbor List this Verlet Container uses.
