Detailed Description

template<class Particle, class CRTP_T>
class autopas::Functor< Particle, CRTP_T >

Functor class.

This class describes the pairwise interactions between particles. Both an array of structure (AoS) and a structure of array (SoA) are supported to be used with functors. Newton3: A functor does not have to implement both a newton3 and a non-newton3 version. Instead you can specify, which version you use by overriding allowsNonNewton3 resp. allowsNewton3

template<class Particle , class CRTP_T >
virtual bool autopas::Functor< Particle, CRTP_T >::allowsNewton3 ( )
pure virtual

Specifies whether the functor is capable of Newton3-like functors.

If the functor provides an interface to soa or aos functions that utilize Newton's third law of motion (actio = reactio) to reduce the computational complexity this function should return true. If this is not the case this function should return false.

Returns
true if and only if this functor provides an interface to Newton3-like functions.
