Detailed Description

template<class Particle, class ParticleCell, bool modifiable>
class autopas::internal::VerletClusterCellsParticleIterator< Particle, ParticleCell, modifiable >

ParticleIterator class to access particles inside a VerletClusterCells container efficiently.

The particles can be accessed using "iterator->" or "*iterator". The next particle using the ++operator, e.g. "++iterator".

Template Parameters
ParticleType of the particle that is accessed.
ParticleCellType of the container
