Detailed Description

template<class ParticleCell, class PairwiseFunctor, DataLayoutOption::Value dataLayout, bool useNewton3>
class autopas::LCC04CombinedSoATraversal< ParticleCell, PairwiseFunctor, dataLayout, useNewton3 >

This class provides the c04 traversal.

The traversal uses the c04 base step performed on every single cell. Since these steps overlap a domain coloring with eight colors is applied.

Template Parameters
ParticleCellthe type of cells
PairwiseFunctorThe functor that defines the interaction of two particles.
