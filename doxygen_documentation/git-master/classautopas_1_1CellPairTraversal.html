Detailed Description

template<class ParticleCell>
class autopas::CellPairTraversal< ParticleCell >

A cell pair traversal.

This class handles traversals through the cell structures. Derived classes handle the order through which the cells are traversed.

Template Parameters
ParticleCelltype of cells.
