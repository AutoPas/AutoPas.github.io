Detailed Description

template<class ParticleCell, class PairwiseFunctor, int collapseDepth = 3>
class autopas::CBasedTraversal< ParticleCell, PairwiseFunctor, collapseDepth >

This class provides the base for traversals using base steps based on cell coloring.

Template Parameters
ParticleCellthe type of cells
PairwiseFunctorThe functor that defines the interaction of two particles.
collapseDepthSet the depth of loop collapsion for OpenMP. Loop variables from outer to inner loop: z,y,x
