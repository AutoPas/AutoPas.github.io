Detailed Description

template<class ParticleCell>
class autopas::internal::CellBlock3D< ParticleCell >

Class that manages a block of ParticleCells.

It is used to resize the cellblock and to handle the conversion of 3d to 1d indices

Template Parameters
ParticleCelltype of the handled ParticleCells

inline

Get the nearby halo cells.

A list of halo cells is returned whose distance to position is at most allowedDistance. If position is inside a halo cell that cell is also returned. (1 norm is used, i.e. the distance is computed for each dimension separately, Manhattan distance)

Parameters
positioncells close to this position are to be returned
allowedDistancethe maximal distance to the position
Returns
a container of references to nearby halo cells
