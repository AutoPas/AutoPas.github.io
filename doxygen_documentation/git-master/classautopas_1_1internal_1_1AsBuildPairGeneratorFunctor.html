Detailed Description

template<class Particle, bool callCheckInstead = false>
class autopas::internal::AsBuildPairGeneratorFunctor< Particle, callCheckInstead >

This functor can generate or check variable verlet lists using the typical pairwise traversal.

Template Parameters
ParticleThe particle class.
callCheckInsteadIf false, generate a neighbor list. If true, check the current for validity. Checking validity only works with the AoSFunctor().
