Detailed Description

template<class ParticleCell, class ParticleFunctor, bool bidirectional = true>
class autopas::internal::CellFunctor< ParticleCell, ParticleFunctor, bidirectional >

A cell functor.

This functor is built from the normal Functor of the template type ParticleFunctor. It is an internal object to handle interactions between two cells of particles.
