Detailed Description

template<class Particle, class ParticleCell, class ParticleFunctor, DataLayoutOption::Value DataLayout, bool useNewton3 = true, bool bidirectional = true>
class autopas::internal::CellFunctor< Particle, ParticleCell, ParticleFunctor, DataLayout, useNewton3, bidirectional >

A cell functor.

This functor is build from the normal Functor of the template type ParticleFunctor. It is an internal object to handle interactions between two cells of particles.
