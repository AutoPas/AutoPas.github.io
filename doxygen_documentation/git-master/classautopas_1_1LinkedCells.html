Detailed Description

template<class ParticleCell, class SoAArraysType = typename ParticleCell::ParticleType::SoAArraysType>
class autopas::LinkedCells< ParticleCell, SoAArraysType >

LinkedCells class.

This class uses a list of neighboring cells to store the particles. These cells dimensions are at least as large as the given cutoff radius, therefore short-range interactions only need to be calculated between particles in neighboring cells.

Template Parameters
ParticleCelltype of the ParticleCells that are used to store the particles
SoAArraysTypetype of the SoA, needed for verlet lists
