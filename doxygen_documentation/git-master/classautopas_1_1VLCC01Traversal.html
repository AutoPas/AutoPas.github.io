Detailed Description

template<class ParticleCell, class PairwiseFunctor, class NeighborList>
class autopas::VLCC01Traversal< ParticleCell, PairwiseFunctor, NeighborList >

This class provides the c01 traversal.

The traversal uses the c01 base step performed on every single cell. newton3 cannot be applied!

Template Parameters
ParticleCellthe type of cells
PairwiseFunctorThe functor that defines the interaction of two particles.
NeighborListtype of the neighbor list
