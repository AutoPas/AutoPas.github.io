Detailed Description

template<class Particle, class ParticleCell, bool modifiable>
class autopas::internal::ParticleIterator< Particle, ParticleCell, modifiable >

ParticleIterator class to access particles inside of a container.

The particles can be accessed using "iterator->" or "*iterator". The next particle using the ++operator, e.g. "++iterator".

Template Parameters
ParticleType of the particle that is accessed.
ParticleCellType of the cell of the underlying data structure of the container.
modifiableDefines whether the ParticleIterator is modifiable or not. If it is false, it points to a const Particle.

Constructor of the ParticleIterator class.

When started in a parallel block iterators are generated with offset and iterate in a round robin fashion. If there are more threads than cells surplus iterators are directly set to cont->end().

Parameters
contLinear data vector of ParticleCells.
offsetNumber of cells to skip before starting to iterate.
flagManagerThe CellBorderAndFlagManager that shall be used to query the cell types. Can be nullptr if the behavior is haloAndOwned.
behaviorThe IteratorBehavior that specifies which type of cells shall be iterated through.
