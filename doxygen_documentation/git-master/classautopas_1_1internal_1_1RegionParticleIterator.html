Detailed Description

template<class Particle, class ParticleCell, bool modifiable>
class autopas::internal::RegionParticleIterator< Particle, ParticleCell, modifiable >

RegionParticleIterator to iterate over all particles within a specific region.

Template Parameters
ParticleParticle type over which the iterator iterates
ParticleCellCell type over which the iterator iterates
modifiableDefines whether the ParticleIterator is modifiable or not. If it is false, it points to a const Particle.
