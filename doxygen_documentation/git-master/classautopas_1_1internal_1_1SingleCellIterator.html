Detailed Description

template<class Particle, class ParticleCell, bool modifiable>
class autopas::internal::SingleCellIterator< Particle, ParticleCell, modifiable >

SingleCellIterator class to loop over particles of a single cell.

Template Parameters
Particletype of the Particles
ParticleCelltype of the ParticleCell.
modifiableDefines whether the ParticleIterator is modifiable or not. If it is false, it points to a const Particle.
