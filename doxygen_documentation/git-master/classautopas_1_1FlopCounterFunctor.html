Detailed Description

template<class Particle>
class autopas::FlopCounterFunctor< Particle >

This class helps in getting the number of performed floating point operations.

It is a functor that only calculated the amount of floating point operations.

Todo:
this class currently is limited to the following case:
  • constant cutoff radius
  • constant amount of floating point operations for one kernel call (distance < cutoff)

Specifies whether the functor is capable of Newton3-like functors.

If the functor provides an interface to soa or aos functions that utilize Newton's third law of motion (actio = reactio) to reduce the computational complexity this function should return true. If this is not the case this function should return false.
