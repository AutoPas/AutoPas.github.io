Detailed Description

template<class Particle, class ParticleCell>
class autopas::AutoTuner< Particle, ParticleCell >

Automated tuner for optimal iteration performance.

This class offers an interface to the iteratePairwise method. Internally it chooses the best container, traversal etc for the current simulation.

Save the runtime of a given traversal if the functor is relevant for tuning.

Samples are collected and reduced to one single value according to _selectorStrategy. Only then the value is passed on to the tuning strategy. This function expects that samples of the same configuration are taken consecutively. The time argument is a long because std::chrono::duration::count returns a long.
