Detailed Description

template<class ParticleCell, class PairwiseFunctor>
class autopas::LCC18Traversal< ParticleCell, PairwiseFunctor >

This class provides the lc_c18 traversal.

The traversal uses the c18 base step performed on every single cell.

C18 base step in 2D. (dark blue cell = base cell)

Since these steps overlap a domain coloring with eighteen colors is applied.

C18 domain coloring in 2D. 6 colors are required.
Template Parameters
ParticleCellthe type of cells
PairwiseFunctorThe functor that defines the interaction of two particles.
