Detailed Description

template<class T, int getOptChar>
struct MDFlexConfig::MDFlexOption< T, getOptChar >

Struct to bundle information for options.

Template Parameters
TDatatype of the option
getOptCharint for the switch case that is used during cli argument parsing with getOpt.
Note
ints should be unique so they can be used for a switch case.
getOptChar should never be -1 because getopt uses this value to indicate that there are no more cli arguments
use LINE as a cheap generator for unique ints.
